# ReTube

## О проекте
ReTube - это сервис для публикации видеороликов, разработанный с использованием **ASP.NET CORE Web API**
*В репозитории есть решение `.sln` с двумя проектами: основной `ReTube` и с тестами`ReTube.Tests`.*

## Функционал
- Регистрация и аунтефикация пользователей (JWT)
- Создание, просмотр, обновление и удаление видеороликов
- Возможность создавать и удалять свои плейлисты
- Добавлять и удалять из плейлистов видеоролики
- Возможность добавлять любимые видео в избранные или удалить их оттуда

## Технологии
- C#
- ASP.NET CORE Web API
- Entity Framework Core
- Microsoft SQL Server
- Identity и JWT

## Как запустить проект
### Склонируем репозиторий
```bash
   git clone https://github.com/akanelovw/TaskTracker.git
```
- Настраивем подключение к базе данных (В appsettings.json установить корректную строку подключения к SQL Server.)
### Применим миграции
```bash
   dotnet ef database update
```
- Запускаем проект
